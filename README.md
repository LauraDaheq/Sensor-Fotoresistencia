# Arduino LDR Light Sensor

## Overview
This project implements an ambient light detection system using an Arduino Uno and a photoresistor (LDR) sensor module. The Arduino continuously reads the light intensity and controls an LED based on a predefined threshold.

## Features
- Ambient light detection
- Analog sensor reading
- Real-time LED control
- Adjustable sensitivity through the sensor module potentiometer
- Easy-to-build electronic circuit

## Components
- Arduino Uno
- LDR Sensor Module
- LED
- 220 Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

## Software
- Arduino IDE

## Working Principle
The LDR sensor measures the surrounding light intensity and sends an analog signal to the Arduino Uno. The microcontroller processes the sensor value and determines whether to turn the LED on or off according to the programmed threshold.

## Applications
- Automatic lighting systems
- Smart home projects
- Light monitoring
- Educational Arduino projects
- Embedded systems learning
